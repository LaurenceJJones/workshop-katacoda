## CrowdSec WAF

You may see the CrowdSec WAF referenced as AppSec or Application Security Component in this workshop and documentation.

In this workshop, you will learn how to setup the CrowdSec WAF and how to configure NGINX to work with the component.

### What is CrowdSec WAF ?

The CrowdSec WAF is a component that can be used to protect your web applications. It allows web servers to forward the request to the component and get a response back to allow or block the request.
